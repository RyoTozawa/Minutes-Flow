# Minutes-Flow
現在、大学で1年間を通して、開発をしています。
現在、議事録作成は、docxで作成しPDFに変換し、メールにて共有するというフローになっています。
このプログラムは、Markdownで、議事録を書きPushすると、Webhookで、mdをPDFに変換し、メールで共有してくれるものです。
